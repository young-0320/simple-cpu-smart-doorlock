0. 프로젝트 구조 간단 소개
지금부터 저희가 구현한 스마트 도어락 기능을 시연하겠습니다.
숫자는 PMOD를 이용해 one-hot 방식으로 입력받고, 입력된 숫자에 따라 도어락 열림, 입력 취소, 오류 횟수 표시, 입력 지연, 마스터키, 비밀번호 변경 기능이 동작합니다.
코드 구조와 CPU 동작 방식은 발표에서 따로 설명하고, 영상에서는 실제 기능 동작 위주로 보여드리겠습니다.

1. 초기 상태 확인
먼저 보드를 리셋하여 초기 상태를 확인하겠습니다.
리셋 후 아무 LED도 켜져 있지 않은 상태가 도어락이 닫혀 있는 기본 상태입니다.
이 상태에서 숫자 입력이 들어오면 도어락 동작이 시작됩니다.

2. 초기 비밀번호 입력으로 도어락 열림 확인
먼저 초기 비밀번호를 입력해보겠습니다.
현재 초기 비밀번호는 1, 2, 3, 4로 설정되어 있습니다.
비밀번호가 정상적으로 입력되면 Open LED가 켜지면서 도어락이 열린 상태가 됩니다.
이후 실제 문을 닫는 상황을 reset 버튼으로 표현하여 닫힘 상태를 보여드리겠습니다. 보시는 바와 같이 Open LED를 꺼지게 되고 이는 닫힘 상태가 되었음을 알려줍니다.

3. 입력 흐름 중 비밀번호 포함 여부 확인
다음은 입력 흐름 중 비밀번호 포함 여부를 확인하겠습니다.
이 도어락은 전체 입력값이 비밀번호와 완전히 같지 않아도 됩니다.
입력된 숫자 흐름 중에서 저장된 비밀번호 4자리가 연속으로 포함되면 도어락이 열리도록 구현했습니다.
따라서 5, 6을 먼저 입력하고, 그 뒤에 1, 2, 3, 4가 연속으로 입력한 뒤, 추가 7, 8을 입력하여도 Open LED가 켜집니다.

4. 입력 취소 기능 확인
이번에는 입력 취소 기능을 확인하겠습니다.
먼저 1, 2를 입력한 뒤 잘못된 숫자인 9를 입력합니다.
이때 취소 버튼을 누르면 가장 마지막에 입력된 9가 삭제됩니다.
이후 3과 4를 이어서 입력하면 최종 입력은 1, 2, 3, 4가 되므로 도어락이 정상적으로 열립니다.

5. 비밀번호 오류 1회 확인
다음은 비밀번호 오류 횟수에 따른 LED 표시를 확인하겠습니다.
먼저 틀린 비밀번호를 한 번 입력합니다.
비밀번호가 일치하지 않으면 실패 횟수가 1회로 증가하고, 6번 LED에 빨간색 LED가 점등됩니다.

6. 비밀번호 오류 2회 확인
다시 한 번 틀린 비밀번호를 입력하겠습니다.
두 번째 오류가 발생하면 실패 횟수가 2회로 증가하고, LED 5번과 6번에 모두 빨간색 LED가 점등됩니다.
이를 통해 연속 오류 횟수가 누적되고 있음을 확인할 수 있습니다.

7. 비밀번호 오류 3회 및 입력 지연 확인
세 번째로 틀린 비밀번호를 입력하겠습니다.
연속 오류가 3회가 되면 기존 Red LED에 추가로 외부 Red LED까지 켜져 총 3개의 LED가 점등됩니다.
이 상태에서는 보안 기능으로 약 10초 동안 입력 제한이 걸립니다.
지금 숫자를 입력해도 도어락이 반응하지 않는 것을 확인할 수 있습니다.
2분이 지나면 입력 제한이 해제되고 다시 비밀번호 입력이 가능해집니다.

8. 마스터키 입력으로 즉시 열림 확인
다음은 마스터키 기능입니다.
마스터키는 일반 비밀번호와 별도로 저장된 번호입니다. 이때 저희는 9999를 마스터키로 저장해두었습니다.
입력된 숫자가 마스터키와 일치하면 일반 비밀번호 입력 과정과 관계없이 즉시 도어락이 열립니다.
마스터키를 입력하면 Open LED가 켜지는 것을 확인할 수 있습니다.

9. 비밀번호 변경 모드 진입
이번에는 비밀번호 변경 기능을 시연하겠습니다.
스위치를 올리면 도어락이 비밀번호 변경 모드로 전환됩니다.
비밀번호는 4자리에서 8자리까지 설정할 수 있으며, 이번 시연에서는 8자리 비밀번호로 변경해보겠습니다.
먼저 새 비밀번호 8자리(12345678)를 입력하고, 확인을 위해 같은 비밀번호를 한 번 더 입력합니다. 두 입력이 일치하면 새 비밀번호로 변경됩니다.

10. 변경된 비밀번호로 도어락 열림 확인
비밀번호 변경이 정상적으로 되었는지 확인하겠습니다.
먼저 기존 비밀번호였던 1, 2, 3, 4를 입력합니다.
비밀번호가 변경되었기 때문에 기존 비밀번호로는 도어락이 열리지 않습니다.
이제 방금 설정한 새 비밀번호 8자리(12345678)를 입력하겠습니다.
새 비밀번호가 정상적으로 입력되면 Open LED가 켜지면서 도어락이 열립니다.

11. 마무리 정리
지금까지 스마트 도어락의 주요 기능을 시연했습니다.
초기 비밀번호 입력, 입력 흐름 중 비밀번호 포함 판단, 입력 취소, 오류 횟수에 따른 LED 표시, 3회 오류 시 2분 입력 제한, 마스터키 입력, 그리고 비밀번호 변경 기능까지 정상적으로 동작하는 것을 확인했습니다.

이상으로 스마트 도어락 시연을 마치겠습니다.
