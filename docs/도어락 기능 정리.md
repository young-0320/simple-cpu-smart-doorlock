# 도어락 기능 정리

1. 비밀번호 입력 (4자리에서 8자리, one hot 방식(10비트),)
2. 입력 취소 기능 (취소 버튼 누르면 한 자리가 딜리트 되는 형식)
3. 저장된 비밀번호와 일치 여부 판단
4. 일치 시 도어락 열림 신호 출력 -> 열림 신호 출력 후 일정 시간 뒤 닫힘 신호 출력 (이때 열림 지속 시간 상의 필요)
5. 불일치시 LED 점등
6. N회 이상 연속 불일치시 LED 점등 및 비밀번호 입력 대기 시간 적용 ( 비밀번호 입력 대기 시간 상의 필요)
7. 비밀번호 변경 기능 (기존 비밀번호를 누른 뒤 변경 가능, 변경 후 확인 입력 절차)
8. 열림/닫힘 상태 LED 표시
9. 입력 전까지 입력만 기다리는 저전력 모드로 있도록 하다가 비밀번호 입력시 동작 시작하는 식의 기능
